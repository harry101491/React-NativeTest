# React Native Application

A React Native Application that tests components and other features in the IOS and android platforms. 

## Getting Started

Install the create-react-native-app globaly on your computer(Mac Only)

```
npm install -g create-react-native-app 
```
or
```
yarn global add create-react-native-app  
```

After install user either yarn or npm start the application use for mac with XCode only

```
npm run ios
```
or
```
yarn run ios
```
